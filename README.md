<h1>URL directory list searcher</h1>
Its basic level url path discovery tool. I wrote this code for learning some multi-thread coding, like a practice.

<h2>Installation</h2>
<code>git clone https://github.com/NorthNo/Pfrog</code>
<code>cd Pfrog</code>
<code>pip install -r requirements.txt</code>

<h2>Using</h2>
<code>python -u "target url" -w "wordlist you can use default one" -t "Thread count"</code>
